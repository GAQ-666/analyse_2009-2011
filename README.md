# 2009-2011年英国某批发平台销售分析
数据集名称：Online Retail II
来源：UCI Machine Learning Repository（加州大学欧文分校机器学习数据集库）
下载地址：https://archive.ics.uci.edu/dataset/502/online+retail+ii

数据处理阶段
从Excel打开发现包含两张表2009-2010 与 2010-2011 两年的销售数据,表头共九列
从notebook中读取文件后将他们合并为file_r,并且补充表头
将customer_id为空的所有数据分离作为file_1,正常customer_id数据作为file_2

file_2数据清洗阶段
删除file_2中price=0的数据共71行,清洗商品代码为异常的数据5000行左右,剩余正常订单819383行订单数据

file_2分析阶段
1.退货分析
按quantity<0筛选出退货订单17824行
对于这些退货订单分别从: 国家,时间,商品,用户,价格 这五个维度进行退货缘由分析
得到各国退货率及退货金额排行表,并输出结论和建议
得到季节,月份,星期退货率排行表,并输出结论和建议
得到商品退货率表(TOP15),并输出结论建议
从退货次数,退货金额及退货率三方面分析用户退货缘由,输出指标TOP10的用户,得到具体用户的恶意退货行为等等
将消费分为六个价位,分析每个价位的退货数量/退货率/退货占比,输出结论和建议

2.
对file_2进行了以下这些分析
各国家GMV贡献及占比,输出GMV占比/订单数占比/平均客单价三项指标排行,输出结论和建议
各月份GMV贡献及占比,输出GMV占比/订单数占比/平均客单价三项指标排行
商品GMV贡献TOP30（含GMV占比/订单数占比/平均客单价三项指标）,输出结论建议
热门商品（用户数>=500）复购率排行（包含复购率及用户数两项指标）,输出结论建议
热门商品关联分析（TOP20）,分析各商品之间强关联,得到结论和输出建议
RFM用户价值判断及分析,对用户进行RFM三维立体分层,得到五层用户的各项数据
对于各层用户进行画像分析,输出画像及其喜爱商品TOP20,得到结论并输出建议
发现GMV贡献极高的两名客户,对他们深层挖掘,得到各项消费数据,输出建议
对比两个核心贡献层用户的画像,并且输出两层用户的R/F/GMV贡献表（TOP30）,得到两层用户的差距与基础,输出建议
拆解核心层用户发现存在两类用户,对比他们各自的喜爱商品,得到其共同喜爱的商品及各自喜爱商品,输出结论建议
高价值流失及潜力用户的画像与用户数据分析
画出12张核心分析图表

file_1分析阶段
分析file_1过程中发现一些数据quantity与price及商品描述description出现异常情况,并且这些数据的订单代码invoice_id非标准格式
找出quantity及price异常的订单并确认无参考价值,于是将invoice_id非标准格式的订单删除
经过清洗从file_1243007行数据清洗出一万多行异常订单
对file_1这张无customer_id的订单进行了: 国家,时间（季节,月份,星期）,商品等几个维度的退货原因分析并输出结论
除了退货分析以外,还汇总了匿名用户的总GMV以及总订单数平均GMV等等
对匿名用户的订单进行了 销量*金额 二维分析从而找出了匿名用户畅销商品TOP15
并对匿名用户的订单以月份和星期为单位进行了时间销量排行,找出销量最高/最低的时间段
